About

This API allows users to browse a product catalogue, create and manage products, and place orders. Orders automatically validate stock and update inventory. The API is documented with Swagger UI and the project page is hosted on AWS S3.

API Endpoints
Products
GET /products/ List all products
GET /products/{id} Get one product by ID
POST /products/ Create a new product
PUT /products/{id} Update a product
DELETE /products/{id} Delete a product
Orders
GET /orders/ List all orders
GET /orders/{id} Get one order by ID
POST /orders/ Place a new order
PATCH /orders/{id}/status Update order status
